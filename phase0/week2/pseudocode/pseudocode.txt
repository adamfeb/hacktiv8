//
// 1. Newton Second Law
//
// Percepatan sebuah benda akan sebanding dengan gaya yang diberikan pada benda dan berbanding terbalik dengan massa benda. 
// Arah percepatan benda sama dengan arah gaya total yang diberikan pada benda.
//
// Secara matematis hukum II Newton dirumuskan sebagai berikut: ΣF = m x a
// ΣF = resultan gaya (Newton)
// m = massa benda (kg)
// a = percepatan benda (m/s2)
//
// Berdasarkan keterangan di atas, buatlah sebuah algoritma / pseudocode untuk menghitung 
// resultan gaya pada sebuah mobil yang memiliki massa benda 600 kg dan ketika didorong oleh 
// tiga orang percepatannya adalah 2 m/s2!
//


//Algorithm

1. Buat sebuah variable dengan nama "mass" dan berikan nilai sebesar 600
2. Buat sebuah variable dengan nama "acceleration" dan berikan nilai sebesar 2
3. Buat sebuah variable dengan nama "force" tanpa nilai
4. Hitung "mass" dikali "acceleration"
5. Berikan nilai pada "force" sesuai dengan hasil perhitungan
6. Tampilkan nilai "force"


// PseudoCode

STORE "mass" with 600
STORE "acceleration" with 2
STORE "force" without any value

CALCULATE "mass" times "acceleration"
SET "force" value with calculation result

DISPLAY "force"





//
// 2. Tahun Kabisat
//
// Apa yang kamu ketahui tentang tanggal 29 Februari? 
// Kamu pasti tahu jika suatu tahun memiliki tanggal 29 Februari berarti tahun tersebut merupakan tahun kabisat.
//
// Dalam kalender Gregorian, tahun kabisat memiliki beberapa kriteria yaitu antara lain:
//  Jika tahun habis di bagi 4 dan tidak habis di bagi 100, atau
//  Jika tahun habis di bagi 4, habis di bagi 100 dan habis di bagi 400
// Buatlah algoritma & pseudocode untuk menentukan apakah suatu tahun merupakan tahun kabisat atau bukan!
//


// Algorithm

1. Dapatkan nilai dan berikan nilai tersebut ke variable "tahun"
2. Jika sisa hasil bagi dari "tahun" dibagi 4 adalah 0 dan sisa hasil bagi dari "tahun" dibagi 100 adalah 1,
   atau jika sisa hasil bagi dari "tahun" dibagi 4 adalah 0 dan sisa hasil bagi dari "tahun" dibagi 100 adalah 0 dan sisa hasil bagi dari "tahun" dibagi 400 adalah 0
3. Tampilkan "Ini adalah Tahun Kabisat"
4. Selain itu tampilkan "Ini bukan Tahun Kabisat"


// PseudoCode

READ "year"

IF ("year" mod 4 is 0 AND "year" mod 100 is 1) OR ("year" mod 4 is 0 AND "year" mod 100 is 0 AND "year" mod 400 is 0)
  WRITE "This is a Leap Year"
ELSE 
  WRITE "This is not a Leap Year"
ENDIF





//
// 3. Laundry Day
//
// Foxie akan mencuci pakaiannya menggunakan mesin cuci. 
// Pakaian yang akan dicuci oleh Foxie sebanyak 20 dan akan dimasukkan ke mesin cuci. 
// Mesin cuci akan dinyalakan jika semua pakaian Foxie sudah masuk ke mesin cuci.
// 
// Bantulah Foxie untuk menghitung jumlah pakaian yang akan dimasukkan ke mesin cuci menggunakan algoritma & pseudocode perulangan!
//


// Algorithm

1. Buat sebuah variable dengan nama "jumlahPakaian" dan berikan nilai sebesar 20
2. Atur nilai awal "perhitungan" sebesar 0
3. Selama "perhitungan" kurang dari "jumlahPakaian" lakukan
     Tambahkan 1 ke "perhitungan" lalu
     Catat nilai "perhitungan" lalu
     Lakukan "Masukan Pakaian Ke Mesin Cuci"
4. Setelah itu "Nyalakan Mesin Cuci"


// PseudoCode

STORE "clothes" with 20
SET initial "count" to 0

WHILE "count" < "clothes"
  ADD 1 to "count"
  WRITE "count"
  DO "Put Clothes in Washing Machine" 
ENDWHILE

DO "Turn On Washing Machine"





//
// 4. Periksa Kuku
//
// Seorang guru akan memeriksa kuku siswa-siswinya yang sebanyak 40 orang dengan cara berkeliling kelas. 
// Jika guru menemukan siswa/siswi yang memiliki kuku yang panjang maka guru akan menghukum siswa/siswi tersebut,
// jika tidak guru akan memuji siswa/siswi tersebut.
// 
// Buatlah algoritma & pseudocode untuk permasalahan diatas.
//


// Algorithm

1. Buat sebuah variable dengan nama "murid" dan berikan nilai sebesar 40
2. Atur nilai awal "perhitungan" sebesar 0
3. Selama "perhitungan" kurang dari "jumlahMurid" lakukan
     Tambahkan 1 ke "perhitungan" lalu
     Catat nilai "perhitungan" lalu
     Lakukan "Pemeriksaan Kuku"
       Jika Kuku Murid Panjang, Guru menghukum siswa tersebut
       Selain Itu, Guru memberikan pujian



// PseudoCode

STORE "students" with 40
SET initial "count" to 0

WHILE count < students
  ADD 1 to count
  WRITE count
  DO "Check the Student's Nails" 
    IF "Nails is Long"
      DO "Give a Punishment to the Student"
    ELSE
      DO "Give a Compliment to the Student"
    ENDIF
ENDWHILE
